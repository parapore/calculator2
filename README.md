# Name
電卓：答えが3のときだけ・・・


# Features

・シンプルな電卓アプリ。
・答えが3の時だけ画像が表示。
・掛け算割り算の優先計算。
・レスポンシブ対応。


# Requirement

動作環境：Android4.4以上。


# Installation
インストール方法。
Google play storeからダウンロードしてインストールしてください。
[Google play store](https://play.google.com/store/apps/details?id=to.msn.wings.caluculator)
